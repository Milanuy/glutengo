# Auditoria de ubicaciones

## Criterio

Las fichas iniciales pueden venir de informacion publica, comunidad o busquedas manuales.
Cuando haya duda de coordenadas, priorizar:

1. Web oficial o perfil oficial del local.
2. Servicio publico de direcciones de Uruguay: `https://direcciones.ide.uy`.
3. Mapas publicos como Google Maps, Waze u OpenStreetMap.
4. Confirmacion directa con el negocio.
