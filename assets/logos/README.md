# Logos de locales

Guardar aca los logos enviados por locales Verificados o Certificados.

En `data.js`, referenciarlos con:

```js
logoUrl: "/assets/logos/slug-del-local.svg"
```

Usar archivos entregados o autorizados por el negocio. Evitar placeholders genericos para que el beneficio del plan sea identidad real.
